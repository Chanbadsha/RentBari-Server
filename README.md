# Rent-Bari Frontend

Rent-Bari is a modern rental property management platform designed to connect tenants and property owners through a secure, efficient, and user-friendly ecosystem. The platform enables property owners to list and manage rental properties while allowing tenants to discover, save, and book their ideal homes with ease. This repository contains the frontend application of Rent-Bari, built using modern web technologies and best practices.

## About The Project

The Rent-Bari frontend is developed with Next.js and React, focusing on performance, scalability, and an intuitive user experience. The application provides a responsive interface for public visitors, tenants, property owners, and administrators, offering role-based access and dashboard experiences tailored to each user type.

The platform implements server-side rendering, server actions, protected routes, and dynamic data fetching to deliver a fast and secure experience across all devices.

## Key Features

Rent-Bari offers a complete rental management ecosystem with property browsing, advanced search and filtering, user authentication, property booking, payment processing, favorites management, and role-based dashboards.

Tenants can browse verified properties, save favorites, submit booking requests, complete payments, and manage their booking history. Property owners can create and manage listings, monitor booking requests, and track their rental business activities. Administrators have full control over the platform, including user management, property approval workflows, booking management, and system moderation.

## Authentication and Security

The application supports secure email/password authentication alongside Google OAuth login. Role-based authorization ensures that users only have access to features and resources relevant to their permissions. Protected routes, middleware authorization, session validation, and secure server-side operations provide additional layers of security throughout the application.

## Dashboard System

Rent-Bari includes dedicated dashboards for three user roles:

### Tenant Dashboard

The tenant dashboard allows users to browse properties, save favorites, create booking requests, manage payments, and monitor their booking activities.

### Owner Dashboard

Property owners can add, edit, and manage property listings, review booking requests, and track property performance through a dedicated management interface.

### Admin Dashboard

Administrators can manage users, approve or reject property listings, monitor bookings, update user roles, suspend accounts, and oversee platform operations from a centralized dashboard.
